## âš¡ Quick Install

```bash
# Backup current config
mv ~/.config/nvim ~/.config/nvim.backup

# Clone pena.Vim
git clone https://github.com/arxngr/pena.Vim ~/.config/nvim

# Launch Neovim to install plugins
nvim
```

> `lazy.nvim` auto-installs plugins on first launch.
